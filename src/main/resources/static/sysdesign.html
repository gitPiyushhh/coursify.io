Frontend

* I am using HTML/CSS/JS and jQuery for AJAX calls in the frontend. These technologies provide a responsive and interactive user interface.

* The frontend renders the UI throgh HTML files and then through the AJAX calls, we make the communication to our API. The flow is: HTML --> Controller

Backend

* I am using Spring Boot and the REST pattern for the backend. This allows for efficient data processing and communication.

* Controller has methods for every required CRUD operation and added some custom operations. The flow is: Controller --> Desired route method.

Database

* I am using the H2 database. It offers lightweight and in-memory storage for quick development and testing.

* The database credentials and configuration is given in the application.properties. The database creates the new connection evrytime the server is loaded and it uses the JPA repository to perform the SQL querryies.

Authentication

* I am using Spring Security for authentication. It ensures secure access control to protect user data.

* Here we are using the Spring Security for our authentication process, there is a file (SecurityConfig.java) which is responsible for the creating users with the different roles.
